# Campjab

A MUC Jabber bot that archives all the messages and shows them in
realtime on a web frontend (inline images and oembed content supported)

# Deploy

Campjab needs MongoLab plugin some environment variables to work:

```
MONGOLAB_URI=
XMPP_HOST=
XMPP_NICK=
XMPP_PASS=
XMPP_ROOM=
XMPP_USER=
OEMBEDLY_KEY=
BASIC_AUTH_USER=
BASIC_AUTH_PASS=
```
